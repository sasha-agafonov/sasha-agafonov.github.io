Contrail

Implementing an animated contrail was probably the most challenging part of the project. The main idea is that the contrail is represented with spherical particles that spawn from the back of the plane. A particle spawns with some velocity, otherwise the contrail would not look natural. We set the scalar value to a fraction of the plane's speed. Additionally, each particle moves in the same direction as the plane when that particle was spawned. When a particle is spawned, it begins to decrease in size before disappearing completely.
